Scientific Results & Impact

Total citations: 26,100 · H-Index: 43 · Last 5 years: 23,200 citations, H-Index 38 (Google Scholar).

Research Interests

My research activities span a wide range of areas in computational and applied statistics, with a strong focus on methodological development and empirical applications. In particular:

  • Bibliometrics & Science Mapping — Statistical techniques for analysing publication patterns, impact assessment, and systematic literature reviews
  • Text Mining and NLP — Corpus analysis of scientific literature and textual datasets
  • Machine Learning — Tree-based models, ensemble methods, neural networks, clustering, and factorial methods
  • Missing Data Imputation — Recursive, incremental algorithms for imputing missing values in qualitative and quantitative data
  • Computational Statistics — Development of estimation algorithms and numerical procedures
  • Statistical Survey Design — Design and execution of socio-economic surveys and customer-satisfaction studies

Scientific Software

  • Bibliometrix & Biblioshiny — R package and web-app for comprehensive science mapping and bibliometric analysis. 1,350,000+ installations; 200+ community video tutorials
  • TALL — Text Analysis for All — R/Shiny app for NLP: text mining, topic modelling, sentiment analysis
  • E2Tree — Explainable Ensemble Trees: interpreting Random Forest and XGBoost models via single tree-like structures
  • contentanalysis — Tools for extracting and analysing scientific content from PDFs, including citation extraction and reference matching
  • openalexR — R client for the OpenAlex API, facilitating access to open bibliographic metadata for scholarly publications
  • DimensionsR — R interface to the Dimensions API for bibliometric data extraction and analysis
  • PubmedR — R interface to the PubMed API for literature retrieval and analysis
  • PLS-SEM Matlab Toolbox — Matlab/Octave package for Partial Least Squares Structural Equation Models
